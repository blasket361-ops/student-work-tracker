アルバイト時間記録 v013 Excel共有版

CSVではなくExcel（.xlsx）を作成・共有します。limits.csv は同じ場所に置いてください。

初回登録

学生名・学籍番号・クラスは最初に1回だけ入力します。入力内容はこのスマホに自動保存されます。

アルバイト情報（最大3件）

アルバイト名と時給を入力します。時給は数字だけ入力してください。例：1100

一括設定

※「未入力だけをなし」は、すでに「あり」で決定した記録を消しません。

日別上限CSV

管理者が GitHub に limits.csv を置くと、アプリ起動時に自動で読み込みます。日付は 2026/7/1、2026-7-1、2026-07-01 のどれでも読めます。
CSV形式：
date,limit
2026/7/1,28
2026/7/20,40
日別上限CSV：未読込

Excelを共有

共有画面が開いたら、LINE・メール・ファイル保存などを選んでください。
